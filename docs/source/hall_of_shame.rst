The Hall of Shame
=================

This page serves as a historical record of hosts that have failed, disappeared, or exhibited "summer host" behavior. The goal is to educate users on the patterns that lead to these failures.

.. warning::

   This list is for educational purposes. Many of these companies simply vanished, taking user data and money with them. **Always keep off-site backups.**

Historical Patterns
-------------------

.. list-table:: The Lifecycle of a Summer Host
   :widths: 20 80
   :header-rows: 1

   * - Phase
     - Description
   * - **June**
     - Launch with a $5 WHMCS license and a stolen website template. Heavy advertising on Discord.
   * - **July**
     - Massive sales. "Founder" buys a new gaming PC with the revenue instead of paying the server bills.
   * - **August**
     - Upstream provider cuts them off for non-payment. Discord server is deleted. "Founder" goes back to school.
   * - **Result**
     - Total loss of world files, player data, and pre-paid balances.

.. tip::

   If you've been burned by a host on this list, consider contributing your story (with proof) to our community repository.
